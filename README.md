# Parser Scripts

## CSV parser
To run the parser script examples for CSV --> JSON, in the terminal be in the 'lesson-02' directory and then write:
``` 
npm run csvParser
```
___

## YAML parser
To run the parser script examples for YAML --> JSON, in the terminal be in the 'lesson-02' directory and then write:
``` 
npm run yamlParser
```
___

## TXT parser
To run the parser script examples for txt --> JSON, in the terminal be in the 'lesson-02' directory and then write:
``` 
npm run txtParser
```
___
